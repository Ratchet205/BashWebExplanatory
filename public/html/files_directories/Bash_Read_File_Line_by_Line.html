Bash Read File Line by Line
Using read command and while loop
Wir können den read-Befehl verwenden, um den Inhalt einer Datei Zeile für Zeile zu lesen. Wir verwenden das -r-Argument für den read-Befehl, um Backslash- Escaped-Zeichen zu vermeiden.

    
    file="temp.txt"

    while read -r line; 
    do 
        echo -e "$line\n"
    done <$file



Using cat and for loop
Von diesem Ansatz aus können wir den Inhalt einer Datei mit dem cat-Befehl und der for-Schleife lesen. Wir verwenden für eine Schleife den gleichen Ansatz wie für die while-Schleife, aber wir speichern den Inhalt der Datei in einer Variablen mit dem cat-Befehl. Die for-Schleife iteriert über die Zeilen in der Ausgabe des cat-Befehls eine nach der anderen, bis sie EOF oder das Ende der Datei erreicht.

    
    file=$(cat temp.txt)

    for line in $file 
    do 
        echo -e "$line\n"
    done
