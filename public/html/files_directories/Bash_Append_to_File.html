Bash Append to File
Append to a File using the Redirection Operator (>>)
Mit der Umleitung können Sie die Ausgabe eines Befehls erfassen und als Eingabe an einen anderen Befehl oder eine andere Datei senden. Der Operator >> edirection hängt die Ausgabe an eine gegebene Datei an. Es gibt eine Reihe von Befehlen, mit denen Sie Text auf die Standardausgabe drucken und in die Datei umleiten können, wobei echo und printf die am häufigsten verwendeten sind.

    
    $ echo "this is a new line" >> file.txt



Bei Verwendung mit der Option -e interpretiert der echo-Befehl die umgekehrten Schrägstriche wie z.B. newline \n:

    
    $ echo -e "this is a new line \nthis is another new line" >> file.txt



Um komplexere Ausgaben zu erzeugen, verwenden Sie den Befehl printf, mit dem Sie die Formatierung der Ausgabe festlegen können:

    
    $ printf "Hello, I'm %s.\n" $USER >> file.txt



Append to a File using the tee Command
tee ist ein Befehlszeilenprogramm unter Linux, das von der Standard-Eingabe liest und sowohl in die Standard-Ausgabe als auch in eine oder mehrere Dateien gleichzeitig schreibt. Standardmäßig verwritet der Befehl tee die angegebene Datei. Um die Ausgabe an die Datei anzuhängen, verwenden Sie tee mit der Option -a (--append):

    
    $ echo "this is a new line" | tee -a file.txt



Wenn Sie nicht möchten, dass tee auf die Standardausgabe schreibt, leiten Sie sie nach /dev/null um:

    
    $ echo "this is a new line" | tee -a file.txt >/dev/null



Um Text an mehr als eine Datei anzuhängen, geben Sie die Dateien als Argumente für den tee-Befehl an:

    
    $ echo "this is a new line" | tee -a file1.txt file2.txt file3.txt
