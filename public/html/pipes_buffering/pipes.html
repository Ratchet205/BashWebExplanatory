Pipe
Pipes und Buffering sind Konzepte, die oft in der Programmierung, insbesondere bei der Arbeit mit Prozessen und Datenströmen, verwendet werden.
Eine Pipe ist eine Verbindung zwischen zwei Prozessen, über die Daten sequenziell von einem Prozess zum anderen übertragen werden können.
Hier ist ein Beispiel, wie man eine Pipe in einem Bash-Skript erstellt:

    # Erstelle eine Pipe
    mkfifo mypipe
    # Schreibe Daten in die Pipe von einem Prozess
    echo "Hallo, Welt!" > mypipe
 
    # Lese Daten aus der Pipe in einem anderen Prozess
    cat mypipe
    



    • Die Zeile mkfifo mypipe erstellt eine Pipe mit dem Namen "mypipe".
    

    • Mit echo "Hallo, Welt!" > mypipe wird der Text "Hallo, Welt!" in die Pipe geschrieben.
    

    • Der Inhalt der Pipe wird mit cat mypipe ausgegeben.
